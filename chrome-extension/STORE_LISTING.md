# Chrome Web Store 申請用テキスト

## 拡張機能名
note → X 自動拡散ツール

## 短い説明（132文字以内）
noteの記事を開くだけでXに自動投稿。APIキー不要・完全無料。記事の導入部を自動取得して3秒後に投稿まで完結。

## 詳細説明

### こんな悩みを解決します
- noteを書いても誰にも読まれない
- Xで宣伝したいけど毎回手動は面倒
- X APIは高くて使えない

### この拡張でできること
✅ noteの記事を開くだけで自動でXに投稿
✅ 記事の導入部（冒頭150字）を自動取得してツイート文を生成
✅ APIキー不要・月額費用0円
✅ 同じ記事の2重投稿を自動防止
✅ 投稿前に3秒のカウントダウンで確認可能

### 使い方
1. Chromeに拡張を追加
2. noteの記事ページを開く
3. 自動でXの投稿画面が開き、3秒後に投稿完了

### 対応サイト
- note.com
- X（twitter.com / x.com）

### プライバシーについて
- 収集するデータ：投稿済み記事URLのみ（ローカル保存）
- 外部サーバーへの送信：なし
- ログイン情報の取得：なし

## スクリーンショット用の説明
1. noteの記事ページでポップアップを開いた画面

## 申請手順
1. https://chrome.google.com/webstore/devconsole にアクセス
2. Googleアカウントでログイン
3. 初回のみ開発者登録料 $5 を支払う
4. 「新しいアイテム」→ ZIPファイルをアップロード
5. 上記のストア掲載情報を入力
6. スクリーンショットを4枚以上追加
7. 審査に提出（通常1〜3営業日）

## ZIPの作り方（Mac）
chrome-extensionフォルダを右クリック →「圧縮」

## ZIPの作り方（Windows）
chrome-extensionフォルダを右クリック →「送る」→「圧縮（zip形式）フォルダー」
