## 投稿①【朝6-9時｜共感型】

```
既読がついた。

でも、返事がこない。

スマホ、何回見ましたか。

鳴ってないのに、わかってるのに、また見てる。

その時間、彼の頭の中では何が起きてるのか。

「無視してやろう」なんて思ってる男は、ほとんどいません。

それより、ずっとめんどくさい状態になってる。

→ プロフのnoteに書きました。¥300です。
```

## 投稿②【夜18-21時｜教育型】

```
既読スルーが続くほど、
男の罪悪感が積み上がっていく。

「今更なんて返せばいい」
「また傷つけたら嫌だ」

この重さが限界になると、
返したいのに返せなくなる。

だから追いLINEを送るほど、
返ってこなくなる。

仕組みを知ってると、
次の動き方が変わります。

→ プロフのnoteで解説してます。
```

## 投稿③【昼｜問いかけ型】

```
既読スルーされてる時、
あなたはどう過ごしてますか。

ひたすら待つ。
追いLINEを検討する。
最悪の結末を想像する。

どれも消耗するだけです。

彼の心理の仕組みを知ると、
「今追うべきか、待つべきか」が
自分で判断できるようになります。

¥300のnoteに書きました。
→ プロフから。
```

## 使い方

- 投稿①：月・木（朝枠）
- 投稿②：火・金（夜枠）
- 投稿③：水・土（昼枠）

初速（最初の48時間）に投稿①と②を連続で出すと購買率が上がる。
