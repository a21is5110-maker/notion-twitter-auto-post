## Xで拡散するツイートテンプレート

### 復縁記事用
```
「もう無理かも」と思ってた私に、タロットがこう言った。

あの人の本音と、復縁を引き寄せる3つの行動。
全部noteに書きました。

👇
[noteリンク自動挿入]

#復縁 #タロット占い #恋愛相談
```

### 月次占い用
```
【7月の恋愛運】

今月、恋が動く星座・待つ星座を全員分占いました。

あなたの星座は何位？
今月「動くべき日」も全部書いてます👇

[noteリンク自動挿入]

#星座占い #恋愛運 #7月運勢
```

```
彼が自然と「会いたい」と思う女性と

[noteリンク自動挿入]
```

## Chrome拡張の活用法

記事をnoteにアップしたら、記事ページを開くだけで上記ツイートが自動投稿される。
毎日手動でツイートする手間がゼロになり、継続的な流入が確保できる。
