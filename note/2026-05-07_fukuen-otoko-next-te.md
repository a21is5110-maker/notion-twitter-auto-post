## 謝った後に送っていい言葉・送ってはいけない言葉

**送っていい言葉**

```
「返事しなくていいよ。伝えたかっただけだから。」
```

これだけでいい。

これが届いた瞬間、彼の中の義務感が消える。

義務感が消えると、自分の意志で返信するかどうかを考え始める。

それが、復縁の入り口です。

---

**送ってはいけない言葉**

```
「ちゃんと伝わったかな」
「今どんな気持ちか教えてほしい」
「無視しないでほしい」
「返事がないのが一番つらい」
「もう一回だけ話せない？」
```

どれも、相手に「返信しなきゃいけない重さ」を乗せる言葉。

謝りを重ねるのも同じです。

謝れば謝るほど、彼は何と返せばいいかわからなくなっていく。

## 彼が返信したくなる余白の作り方

謝った後、3〜5日は何もしない。

これが基本です。

「何もしない」は諦めじゃない。

「あなたのペースで決めていい」という信号を出している状態。

---

もし5日経っても返信がない場合、

「返事しなくていいよ」の後に、

復縁の話を一切しないゆるい一言を入れる。

例：

```
「そういえば、○○のお店できたみたい。
 一緒に行ってた場所の近くだったから。」
```

責めない。答えを求めない。

ただ、あなたがそこにいることだけを届ける。

これだけで、空気が変わることがある。

## 次に接触するタイミングの見極め方

謝った後の時間で、状況の意味が変わります。

**返信が来た場合**

→ 内容より「温度」を見る。

絵文字があるか。文が短いか長いか。

冷たくないかどうかを確認してから、次を考える。

**3日以内に既読になった場合**

→ 読んでいる。考えている。

まだ動ける距離感です。焦らなくていい。

**既読がついていない場合**

→ まだ見られていないか、気持ちの整理中。

この段階でさらに送るのは逆効果です。

**返信が来たが、短い場合**

→ 嫌悪感はない。でも気持ちが固まっていない。

追わず、少し間を置いてから次のゆるい接触を入れる。
