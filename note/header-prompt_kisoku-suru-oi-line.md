# noteヘッダー画像プロンプト完全版
## 記事：既読スルーの夜、追いLINEを送る前に読むnote
## 作成日：2026年5月3日

---

## 使い方

1. 下記「ChatGPT DALL-Eプロンプト」をコピーしてChatGPTに貼り付ける
2. 画像が生成されたらダウンロード
3. Canvaに取り込み、テキストオーバーレイを追加
4. 1280×670pxで書き出してnoteにアップ

---

## ChatGPT DALL-Eプロンプト

```
Create a note article header image (1280x670px).

Concept: A woman alone late at night, staring at her smartphone screen showing a "read" notification with no reply.

Visual elements:
- Dark midnight blue and black gradient background
- Soft glow from smartphone screen illuminating her face
- Crescent moon or city night lights visible faintly in background
- Minimalist Japanese aesthetic
- Melancholic, quiet, cinematic atmosphere

Style: Soft illustration or cinematic photography style, muted colors, emotional depth

Text area: Leave upper or lower 30% of image clean for text overlay

No text in the image. No people's faces shown clearly. 4K quality.
```

## Canvaデザイン指示

| 項目 | 設定 |
|---|---|
| サイズ | 1280×670px |
| 背景 | 深夜の紺〜黒グラデーション |
| テキスト色 | 白 または #F5E6C8（薄いゴールド） |
| フォント | Noto Serif JP（細め） |
| テキスト位置 | 画像上部または下部30% |
| アクセント | スマホの画面光、三日月 |
| 雰囲気 | 静か・夜・切ない |
| 装飾 | 最小限 |

## 注意事項

- 生成後は必ずCanvaでテキストを追加してから使用

- 明るすぎる・カラフルすぎる場合は「darker, more melancholic」を追記して再生成
- note推奨サイズ：1280×670px（16:8.5比率）
