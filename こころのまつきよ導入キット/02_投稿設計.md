# 02_投稿設計

## Threads基本ルール

- 1投稿1テーマ、1投稿1メッセージ
- 初期は短文80%・長文20%（フォロワーが伸びたら文字数を増やす）
- 朝6:00〜9:00を重視、夜18:00〜21:00を最重視
- 直リンクは避ける。原則プロフリンク誘導

---

## 投稿フォーマット（通常の恋愛不安系）

```
魂の叫び
↓
誰向けか
↓
バズフレーズ
↓
悩み
↓
見立て
↓
鑑定導線
```

**魂の叫び例：**
心折れた〜‼️ / もう無理〜‼️ / 既読で折れた‼️ / 未読で沈んだ‼️
返信なくて崩れた‼️ / 泣きたい夜だ‼️ / 胸がズタズタ‼️

---

## 夜鑑定定型文

```
心折れた〜‼️
返信ない夜は追わずに整えて。
恋の方位、今夜見ます。
夜鑑定はプロフリンクから。
```

---

## 吉日投稿

一粒万倍日や吉日は重い叫びを避ける。

主軸：
```
気持ち膨らむ❤️恋の種まき
🌾一粒万倍日
```

## 1日5投稿スケジュール

| 時間 | 役割 |
|---|---|
| 6:00 | 共感・認知 |
| 7:00 | 保存・判断軸 |
| 8:00 | プロフ遷移 |
| 18:00 | 鑑定誘導 |
| 20:00 | 申込導線 |

---

## Facebook投稿ルール

Facebookは信頼構築媒体。Threadsは短文で刺す。Facebookは背景を語って信頼を作る。

| フェーズ | 文字数 |
|---|---|
| 初期 | 80〜150字 |
| 反応あり | 200〜400字 |
| フォロワー増加中 | 500〜800字 |
| 濃い読者あり | 1,000〜1,500字 |
