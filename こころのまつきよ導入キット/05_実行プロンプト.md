# 05_実行プロンプト

---

## 投稿生成（Threads）

```
01_会社の核.md、02_投稿設計.mdを参照して、
Threads投稿を作成してください。

出力：狙い／投稿タイプ／投稿文／導線／改善ポイント

条件：
1投稿1テーマ。短文中心。プロフリンク誘導。
未来断定と不安煽りは禁止。
```

## Facebook投稿生成

```

文字数は以下から選ぶ：

```

## ココナラ商品文生成

```
01_会社の核.md、03_販売導線.mdを参照して、
ココナラ用の商品説明文を作成してください。

出力：商品タイトル／サービス説明／こんな方におすすめ
      鑑定内容／購入後の流れ／注意事項／CTA

条件：レビュー獲得を優先。外部誘導を前提にしない。
      未来断定と不安煽りは禁止。
```
