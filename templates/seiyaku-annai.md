# 成約率最大化｜深掘り案内文テンプレ
## 弱・中・強 3パターン

---

## 【弱パターン】
### 使用場面：初回相談・鑑定に慣れていない・迷っている様子がある

```
読みました。少し整理させてください。

今の状況、感情の問題ではなくて、関係の配置がズレているように見えます。
好きか嫌いかの話ではなくて、今の二人の距離感が合っていないんです。

無料でお伝えできるのは、ここまでです。

「なぜそのズレが起きているのか」「どこから動けばいいのか」は、
もう少し深く見ないと見えない部分があります。

急ぐ必要はありません。
ただ、今の状態のまま時間だけ経つと、整理するタイミングが難しくなることがあります。

もし気になる方は、「鑑定希望」と送ってください。
状況に合うプランをご案内します。
```

---

## 【中パターン】
### 使用場面：悩みが明確・解決意欲がある・背中を押してほしい様子

```
読みました。整理します。

今見えているのは、愛情がないわけではなくて、
タイミングと距離感がズレているということです。

ただ、そのズレがどこから来ているのか、
どう整えれば動き出すのかは、無料ではここまでしか見えません。

このまま放置すると、判断する機会がどんどん後ろにずれていきます。
迷っている時間が長くなるほど、消耗だけが積み上がります。

今整理するか、もう少し待つか。
それはあなたが決めることです。

ただ、今の状態には深掘りが必要だと感じています。

状況によって以下のプランが合います。

・梅（6,980円）：一点だけ確認したい
・竹（9,800円）：関係性の全体を見たい
・松茸（9,800円）：復縁・音信不通・曖昧な関係を整理したい ← 今の状況に近い
・松（19,800円）：根本から人生ごと整理したい

整理されると、迷いが減ります。
動き方が見えます。

気になる方は「松茸希望」または「プラン相談」と送ってください。
```

## 【強パターン】
### 使用場面：悩みが深刻・時間が経過している・決断を迷っている

```
読みました。

今の状態、率直に言います。

感情の問題ではありません。関係の配置がズレています。
愛されていないのではなく、距離感の見方がズレている可能性があります。

そのズレをそのままにしておくと、二つのことが起きます。

一つは、判断すべきタイミングを逃すこと。
もう一つは、正しくない理由で諦めること、または正しくない理由で追いすぎること。

どちらも、今整理すれば防げます。

無料でお伝えできるのはここまでです。

「なぜズレが起きているのか」「どちらに動けばいいのか」は、
個別に見ないと見えない部分です。

プランは以下の通りです。

・梅（6,980円）：一点だけ確認したい方
・竹（9,800円）：関係性の全体像を見たい方
・松茸（9,800円）：今の状況を一段深く整理したい方 ← 多くの方はここ
・松（19,800円）：根本から長期的に整理したい方

整理されると、今夜の行動が変わります。
迷いながら消耗する時間が、減ります。

必要だと思ったら「鑑定希望」と送ってください。
押し売りはしません。必要な方にだけお届けします。
```

---

## 使い分けガイド

| パターン | 使うタイミング |
|---|---|
| 弱 | 初回・様子見・鑑定未経験 |
| 中 | 悩みが明確・解決意欲あり |
| 強 | 悩みが深刻・長期化・迷い大きい |

---

## 共通ルール

- 押し売り禁止：「必要な方にだけ」の一文を必ず入れる
- 説得ではなく納得：読み手の中で決断させる
- 余白を残す：「急ぐ必要はありません」「それはあなたが決めること」
- 具体を出しすぎない：相手の本音・復縁の可能性は有料に残す
- 未来の変化は「整理される・迷いが減る」に留める（断定しない）
