## STEP3：無料診断文

### 目的
安心させつつ「まだ答えがある状態」で止める

### 構成
```
【状況整理】
[状況を客観的に整理。断定しない]

【感情理解】
[その感情は弱さではなく本気の証拠と伝える]

【構造提示（配置・距離・タイミング・言葉の温度）】
[感情の問題ではなく配置の問題と伝える]

【分岐点】
[このまま待つか、整えてから動くか]
[まだ答えがある状態で止める]

ここは、もう少し深く視たい部分があります。
必要でしたら、深掘り鑑定で整理できます。
```

## STEP4：深掘り案内（3強度）

### 8ステップ構造
1. ズレの言語化
2. 誤解の解除（愛情不足ではない・配置の問題）
3. 放置リスク（柔らかく）
4. 分岐点
5. 有料の必要性（無料の限界）
6. プラン提示
7. 変化イメージ（迷いが減る・動き方が見える）
8. CTA（押し売り禁止・必要な方だけ）

### 弱パターン（初回・未経験）
```

今の状況、感情の問題ではなくて、関係の配置がズレているように見えます。

無料でお伝えできるのは、ここまでです。

急ぐ必要はありません。

気になる方は「鑑定希望」と送ってください。
```

タイミングと距離感がズレているということです。

無料ではここまでしか見えません。
