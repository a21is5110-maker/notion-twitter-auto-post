# 無料診断テンプレ集｜こころのまつきよ
## 注意：答えを出し切らない。未来断定しない。不安を煽らない。

---

## A. 片想い型

**目的：** 感情整理→深掘り誘導
**使用場面：** 「好きだけど何もできない」相談

```
読みました。状況を整理します。

【状況整理】
片想いの相手がいて、気持ちを伝えられない状態が続いている。
相手の言動が気になり、日常生活に影響が出ている。

【感情理解】
好きだから動けない、というのは弱さではありません。
大切に思っているほど、失敗が怖くなります。

【構造提示】
今見えているのは、あなたの気持ちだけです。
相手の気持ちの温度や、今の関係の配置はまだ整理できていない。

【分岐点】
このまま待つか、関係を動かす準備をするか。
どちらが正しいかは、相手との距離感を見てから判断できます。

ここは、もう少し深く視たい部分があります。
必要でしたら、深掘り鑑定で整理できます。
```

**有料に残す余白：** 相手の気持ちの温度・告白タイミング・距離感の縮め方
**深掘り誘導：** 竹または松茸

## B. 復縁型

**目的：** 諦めかけている気持ちを受け止め→松茸誘導
**使用場面：** 「別れたけど忘れられない」相談

```
読みました。状況を整理します。

【状況整理】
別れてから連絡が途絶えている、または散発的な連絡がある状態。
相手のことが忘れられず、新しい恋に踏み出せていない。

【感情理解】
諦められないのは、その恋を本気で大切にしてきた証拠です。
忘れようとしても忘れられない理由が、関係の中にあります。

【構造提示】
今の状態は「終わった恋」ではなく「止まっている恋」に見えます。
ただし、止まっている理由がどこにあるかは、まだ整理できていない。

【分岐点】
今動くべきか、距離を保つべきか。
そのタイミングの見極めが、復縁の分かれ目になります。

ここは、もう少し深く視たい部分があります。
必要でしたら、深掘り鑑定で整理できます。
```

**有料に残す余白：** 復縁の可能性・連絡のタイミング・相手の現在の気持ち
**深掘り誘導：** 松茸（主力）

## C. 音信不通型

**目的：** 孤独感を受け止め→松茸誘導
**使用場面：** 「突然連絡が来なくなった」相談

```
読みました。状況を整理します。

【状況整理】
ある時点から連絡が途絶え、理由もわからない状態が続いている。
自分から連絡すべきか、待つべきか判断できずにいる。

【感情理解】
理由がわからないまま待ち続けるのは、想像以上に消耗します。
それでも待っているのは、その人との関係を大切にしているから。

【構造提示】
音信不通には「意図的なもの」と「状況によるもの」があります。
今の状態がどちらかによって、次の動き方が変わります。

【分岐点】
連絡を取り戻せる可能性があるかどうか。
それを判断するには、関係の配置をもう一段深く見る必要があります。

ここは、もう少し深く視たい部分があります。
必要でしたら、深掘り鑑定で整理できます。
```

**有料に残す余白：** 音信不通の理由・連絡が戻る可能性・最初の一手
**深掘り誘導：** 松茸（主力）

## D. 既読スルー型

**目的：** 追いLINE衝動を止め→竹または松茸誘導
**使用場面：** 「既読がついてから返信が来ない」相談

```
読みました。状況を整理します。

【状況整理】
既読がついているが、返信が来ない状態が続いている。
追いLINEをすべきか迷っている、または送って余計に悪化した経験がある。

【感情理解】
既読スルーひとつで心が削れるのは、その恋を本気で大切にしているから。
これは弱さではなく、本気の証拠です。

【構造提示】
既読スルーは拒絶ではないことがほとんどです。
ただし、スルーの理由と、今の距離感は分けて見る必要があります。

【分岐点】
今追うべきか、距離を整えてから動くべきか。
その判断には、今の関係の配置を見ることが先決です。

ここは、もう少し深く視たい部分があります。
必要でしたら、深掘り鑑定で整理できます。
```

**有料に残す余白：** スルーの理由・追うタイミング・言葉の温度の読み方
**深掘り誘導：** 竹または松茸

## E. 曖昧な関係型

**目的：** 関係の名前のなさを受け止め→松茸誘導
**使用場面：** 「付き合ってないけど離れられない」相談

```
読みました。状況を整理します。

【状況整理】
交際には至っていないが、特別な関係が続いている。
相手の本気度がわからず、このまま続けるべきか迷っている。

【感情理解】
曖昧なまま続けることで消耗しているのに、離れられない。
それは、その関係の中に本物の気持ちがあるからです。

【構造提示】
曖昧な関係には「相手が迷っている」「相手が現状維持を望んでいる」の
大きく2つのパターンがあります。
今どちらにあるかで、次の動き方が変わります。

【分岐点】
このまま待つか、関係を明確にするアクションを取るか。
その判断の前に、相手の気持ちの温度を見る必要があります。

ここは、もう少し深く視たい部分があります。
必要でしたら、深掘り鑑定で整理できます。
```

**有料に残す余白：** 相手の本気度・関係が進む可能性・動くタイミング
**深掘り誘導：** 松茸（主力）

## F. 夫婦・既婚関係型

**目的：** 言えない悩みを受け止め→松鑑定誘導
**使用場面：** 「夫婦関係が冷えた」「不倫関係の整理」相談

```
読みました。状況を整理します。

【状況整理】
長期的な関係の中で、距離感の変化が起きている状態。
誰にも話せず、一人で抱えてきた時間が長い。

【感情理解】
誰にも言えない悩みほど、重く積み上がっていきます。
ここに話してくれたことは、整理するための第一歩です。

【構造提示】
長期的な関係の変化には、表面の出来事だけでなく
関係の構造そのものを見る必要があります。

【分岐点】
今の関係をどう続けるか、あるいは整理するか。
その判断の前に、根本にある部分を確認することが重要です。

ここは、もう少し深く視たい部分があります。
必要でしたら、深掘り鑑定で整理できます。
```

**有料に残す余白：** 関係の根本原因・相手の気持ち・今後の方向性
**深掘り誘導：** 松鑑定（長期・根本）

## G. 仕事・人生相談型

**目的：** 生き方の迷いを整理→松鑑定誘導
**使用場面：** 「転職・独立・人生の方向性が見えない」相談

```
読みました。状況を整理します。

【状況整理】
今の状況に違和感を感じているが、次に何をすべきかわからない状態。
外から見れば問題なさそうでも、内側では答えを探し続けている。

【感情理解】
わからないまま動けないのは、慎重さの表れです。
ただし、その慎重さが長引くほど消耗していきます。

【構造提示】
今見えている選択肢の外に、まだ見えていない可能性があります。
それを整理するには、今の自分の位置を俯瞰して見る必要があります。

【分岐点】
今動くか、もう少し整理してから動くか。
その判断には、現在地の確認が先決です。

ここは、もう少し深く視たい部分があります。
必要でしたら、深掘り鑑定で整理できます。
```

**有料に残す余白：** 具体的な行動指針・タイミング・根本の原因
**深掘り誘導：** 松鑑定（人生全体）

---

## 共通注意事項

- 答えを出し切らない
- 未来を断定しない（「復縁できます」「連絡が来ます」は禁止）
- 不安を煽らない
- 霊的効力を断定しない
- 深層原因・相手の本音・具体行動は有料に残す
- 必ず「分岐点」で止めて深掘り誘導につなげる
