# notion-twitter-auto-post

占い師向けの **Notion → Twitter 自動投稿ツール**。  
Notionに書いた占いコンテンツを定期的にTwitterへ自動投稿し、フォロワーを増やして占い客を集客します。

---

## 機能

- **Notion連携**: データベースの「未投稿」コンテンツを順番に投稿
- **自動集客コンテンツ**: Notionにコンテンツがない時間帯は、今日の運勢・プロモツイートを自動生成して投稿
- **スケジュール投稿**: 設定した間隔（デフォルト3時間）で繰り返し投稿
- **投稿管理**: 投稿済みコンテンツはNotionで「投稿済み」に自動更新

---

## セットアップ

### 1. 依存パッケージのインストール

```bash
pip install -r requirements.txt
```

### 2. 環境変数の設定

```bash
cp .env.example .env
```

`.env` を編集して各APIキーを入力します。

| 変数名 | 説明 |
|---|---|
| `NOTION_TOKEN` | Notion インテグレーションのシークレット |
| `NOTION_DATABASE_ID` | 投稿コンテンツを管理するDBのID |
| `TWITTER_API_KEY` | Twitter Developer Portal で取得 |
| `TWITTER_API_SECRET` | 同上 |
| `TWITTER_ACCESS_TOKEN` | 同上 |
| `TWITTER_ACCESS_TOKEN_SECRET` | 同上 |
| `POST_INTERVAL_HOURS` | 投稿間隔（時間）。デフォルト: `3` |
| `MAX_TWEET_LENGTH` | ツイート最大文字数。デフォルト: `140` |

### 3. Notionデータベースの準備

以下のプロパティを持つデータベースを作成してください。

| プロパティ名 | 種類 | 説明 |
|---|---|---|
| タイトル | タイトル | 管理用タイトル |
| ツイート本文 | テキスト | 実際に投稿する文章 |
| ステータス | セレクト | `未投稿` / `投稿済み` |
| 投稿予定日時 | 日付 | 投稿する順番の優先度 |

セットアップ確認:

```bash
python setup_notion_db.py
```

## 使い方


```bash

```

## Notionコンテンツの書き方

ツイート本文の例:

```
✨ 今週のてんびん座 ✨

人間関係に大きな転機が訪れそうな一週間。
新しい出会いを大切に。
ラッキーカラーは「青」。

▶ 詳しい鑑定はプロフィールのリンクから✨

#占い #てんびん座 #今週の運勢
```

**ポイント:**
- ハッシュタグを含める（`#占い` `#今日の運勢` など）
- CTAを最後に入れる（「DMはこちら」「リンクから予約」など）
- 140文字以内に収める

## 自動生成コンテンツについて

Notionに「未投稿」コンテンツがない場合、以下を自動生成して投稿します。

- **今日の運勢ツイート**: 星座・ラッキーカラー・ラッキーアイテムをランダム生成
- **集客プロモツイート**: 鑑定実績・お客様の声などのPRコンテンツ

`fortune_content.py` のテンプレートを編集することで、自分のスタイルに合わせてカスタマイズできます。
